# fpga-stream-processor

A self-directed Verilog project for simple streaming data processing in RTL.

The goal is to build a small hardware-style pipeline that can take in structured input data, process it through several logic stages, and produce a clear output decision.

This project is intended to improve my understanding of digital design, simulation, verification, and FPGA-style development workflows.

## Status

Proof-of-concept stage.
